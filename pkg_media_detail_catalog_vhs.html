VHS Catalog

Overview

The book catalog contains information indexed by UPC. When uploading to the VHS catalog you must use the manufacturer assigned UPC.

Fields

Field Name Required Description
UPC Y This field must contain a valid UPC or UPC/A barcode value.
Attribute: zoovy:prod_upc
Movie Title Y The title as it appears on the VHS Box
Attribute: zoovy:prod_name
Description Y Text about the VHS
Attribute: zoovy:prod_desc
VHS Cover Y An image of the front of the VHS - minimum: 75x50 maximum: 640x400
Attribute: zoovy:prod_image1
VHS Back Cover N An image of the back of the VHS - minimum: 75x50 maximum: 640x400
Attribute: zoovy:prod_image2
Producer N The person who produced the movie. Last, First OR First Last if more than one producer exists please separate them by ampersands (&)
Attribute: zoovy:prod_producer
Director N The person who directed the movie. Format: Last, First OR First Last if more than one producer exists please separate them by ampersands (&)
Attribute: zoovy:prod_director
Written By N The person who wrote the movie. Last, First OR First Last if more than one writer exists please separate them by ampersands (&)
Attribute: zoovy:prod_writer
Starring N The person(s) who starred in the movie, each on their own line. Format: Last, First OR First Last We recommend not having more then 3 stars per title.
Attribute: zoovy:prod_starring
Cast N Other less well known actors who were in the movie. Please put each person on their own line. Format: Last, First OR First Last
Attribute: zoovy:prod_starring
Genre (Unstructured) N The unstructured genre contains a category for the cd. For example "Pop", "Rock", "Jazz", or "Classical" NOTE: At this time Zoovy is in the process of developing a structured Genre field based on feedback from sellers.
Attribute: zoovy:prod_ugenre
Running Time N The length of the movie in MMM:SS
Attribute: zoovy:prod_runtime
Screen Format N The screen format of the movie.
Attribute: zoovy:prod_screen
Original Release Year N The 4 digit year the movie was originally released in theaters.
Attribute: zoovy:prod_originalyear
Video/VHS Release Year N The 4 digit year the movie was released (or re-released) onto VHS.
Attribute: zoovy:prod_releaseyear
MPAA Rating N Zoovy maintains the list of ratings, please select the MPAA designated rating for the movie.
Attribute: zoovy:prod_rating
Product Language Y The list of available book languages is maintained by Zoovy.
Attribute: zoovy:prod_language
Color N If the movie is in color or black and white.
Attribute: zoovy:prod_color
