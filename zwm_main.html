ZWM Overview

Concept Overview

Web Interfaces are excellent when you make infrequent updates to your website or product database. However if your business is constantly adding and removing products, updating inventory availability, or adding images then it will most certainly experience productivity gains from the benefit of a specialized desktop application.
For recent changes and future directions please review the Doc 50697
The Zoovy warehouse manager is a desktop client intended to simplify the following routine business tasks:
  1. Product Management, see Doc 50832
  2. Image Management, see Doc 50833
  3. Website Category Management, see Doc 50834
  4. Inventory Updates, see Doc 50835

Zoovy Integrated Desktop includes both Zoovy Order Manager and Zoovy Warehouse Manager.

Key Benefits

In addition Warehouse Manager has functionality that cannot be paralleled in a web interface.

Image Cropping, Resizing

The ability to import a variety of images, from different image formats and sources, including built in TWAIN support for scanners or some types of digital cameras. Once an image is imported it can be cropped, or resized, or annotated automatically before uploading to Zoovy.

Management Categories

Management Categories are an internal category structure that you use to keep track of your products. Some features inside Zoovy such as the supply chain module automatically put products into management categories. Management categories are NOT seen by customers on your website. A product can only exist in one Management Category (unlike Navigation categories where a product can exist in multiple categories).

Barcode Printing

Warehouse Manager includes the ability to print out a bar code for your products. This can be used as a compliment to the Package Verification feature found in Zoovy Order Manager.

Enterprise vs. SOHO

The Warehouse Manager desktop client can operate in both a stand alone SOHO mode, and a Multi-User Enterprise configuration. To learn more about this please visit Doc 50836
