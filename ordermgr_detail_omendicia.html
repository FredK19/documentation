What is Endicia?

Overview

Endicia is a 3rd party application which allows you to print USPS Postage. More information is available at http://www.endicia.com and costs approximately $16/mo. Endicia does not mark up the USPS Postage, or charge a fee for "recharging" your account. Of all other methods for printing postage, Endicia is much better suited for the high volume customers Zoovy is known for. Zoovy provides the ability to export orders directly to Endicia in order to print postage. No cutting and pasting between applications, or re-entering data is necessary.
Endicia 4 x 6 labels are supported for both USPS and FedEx. They can be purchased from http://www.endicia.com/Store/Detail/?ID=417
The Endicia Dazzle software is dependent on Internet Explorer. If you are having issues with Internet Explorer Endicia Dazzle may not function correctly.

Stealth Postage

Many sellers who ship with the USPS do so because it's the cheapest way to ship a package. One of the most common reasons for buyer dis-satisfaction and negative feedback on eBay is buyers who feel they were overcharged on shipping. This is because when shipping via the post office the amount you paid for postage is always displayed clearly on the box. Typically the buyer sees that they overpaid on shipping BEFORE THEY EVEN OPEN THE PACKAGE - and needless to say it does not leave a favorable impression. Stealth postage lets you print valid USPS Postage, without a price on it.

Integration

Integration is very easy - just follow the steps below.
'Integrating Zoovy Order Manager:'
  • Select the [Tools] Menu in Order Manager, and then select [Endicia Dazzle Setup]
  • Enter the path to your Endicia Dazzle install, you can either type it in, or browse to the dazzle.exe file.
  • This file is usually installed in: C:\Program Files\Envelope Manager\Endicia\dazzle\dazzle.exe
  • Click on [Save] Your Zoovy Order Manager is now integrated with DaZzle! Please be sure to Restart Order Manager in order for Dazzle to appear in your Right-Click Menu.

    • Importing Tracking Numbers

    Version 4.28 and higher of order manager automatically import tracking numbers when you close Endicia. You can still use the steps below if you plan to keep Endicia open while you are operating Order Manager.

    Printing Shipping Labels/Postage in Zoovy with Dazzle

    In order to print postage, you must have already configured Endicia Dazzle. Open your Dazzle program and set up a layout to match the label or envelope you are printing to. Check all printer settings in Dazzle (File - Printer Setup) and try a TEST PRINT (Print - Test Print) to ensure your piece will print correctly, if this works, save the layout ( File - Save Layout As). Then close or minimize Dazzle.
    'To print postage for a single invoice'
  • Select an invoice in Order Manager, and right click your mouse.
  • Select Endicia DaZzle.
  • A new dialog box will be shown, that will present you with your postage label options for that invoice.
  • The selection box contains a list of available postage types that you can use to print postage.
  • Changing the USPS mail class will change the amount of postage printed once the label is sent to DaZzle.
  • The weight (in Ounces) is copied directly from the invoice.
  • If you wish to manually adjust the weight before printing the label you may do so here.
  • The Ship Value is the declared value of the shipment.
  • This is used primarily for insurance purposes, and affects the value of insurance on the postage printed.
  • Prompt in DaZzle Before Printing- If you wish to confirm each label before it is printed in DaZzle, select this option. This option is on by default. If "No Prompt" is selected, DaZzle will immediately print the postage label once the [Print] button is pressed.
  • [BATCH] - Uses the default settings of the Order Manager to send Endicia DaZzle software

    • Rubber Stamp: For order number to be printed on the USPS shipping label

    Troubleshooting

    Q: Postage Does not Print as Stealth
    A: Make sure you have stealth postage checked in your custom layout. If all else fails, try using one of the standard layouts and see if that works. Zoovy absolutely supports Stealth Postage, so try starting with a known working configuration and re-adding your customization back in.

    To enable stealth mode for particular layout:
    1. Run the Endicia Dazzle software
    2. Open the layout that you are using for Zoovy
    3. Go to Postage -> Options
    4. Click on Premium Service Options
    5. Check Use Key Line (Stealth) Indicia and hit okay
    6. Go to File -> Save Layout and save layout
    7. In Order Manager, go to Tools | Endicia Setup | and make sure "Always Print Stealth Postage" is checked.
    Q: Printing International Postage Corrupts the Layout

    A: Delete the corrupted layout and create a new one.



    Q: The shipping label prints incorrectly, i.e., without postage, barcodes, etc.,

    Did you recently switch your label format? For example, did you recently switch from Domestic Priority mail class to print International Global Priority shipping labels? The problem may be this - The previously selected label format is saved in memory.

    A: Please try the following simple solution:

    1. Open Endicia Dazzle, re-select your preferred/default shipping class, i.e., Domestic Priority Mail, and CLOSE the program.
    2. Answer, "yes" when prompted to save the (changed) format before closing.
    3. Now, print the label again. Right-click the order, choose shipping Labels, and choose Endicia Dazzle Labels.
    4. The shipping label should print fine this time!
    .

    Q: When I try to print Envelope Rate (for those packages weighing less than 3.5 oz), the postage class automatically switches to Package, or some other class not desired.
    A: Since the change in rate classes (as detailed below), many facets of shipping with online postage have also changed, based on the dimensions of the package as well as the weight. Previously used layouts have codes embedded in them that tell Dazzle what rate class to use. So, if you are using a custom layout that was based off of any other layout than the DaZzle Standard, "Envelope", it will likely give this error.

    Steps to make it work:

    1. In DaZzle, open the Envelope standard layout.
    2. Change the media type to suit your needs.
    3. Save the layout with a custom name - something you can recognize.
    4. This will take care of the issue of the postage class changing, and you can print at envelope rate.

    The Envelope Rate Class can ONLY be used if your package is less than .25 inches thick, AND weighs less than 3.5 ounces. Also, this weight class does not permit Delivery Confirmation.


    General Troubleshooting with Custom Labels:

    Custom label layouts settings override Zoovy Order Manager. Since the majority of our customers use a custom label, please review the following if you have trouble printing from Order Manager to Endicia/DaZzle.

    If a label fails to print and a custom layout is being used, please load Endicia and open a new, standard layout from Endicia. Configure the same package geometry (e.g. weight, package dimensions, shipping address) and see if Endicia can print out the label BEFORE contacting Zoovy support.

    When USPS changes their rules and adds new package types, those package types may not be supported by custom layouts and may need to be added.

    This specifically affects settings such as Stealth Postage not printing.

    USPS made significant changes to their mail classes recently. Formerly, the following mail classes were supported: First Class Mail, First Class Postcard Rate, Priority Mail, Priority Mail Flat Rate Env., Priority Mail Flat Rate Box, Package Services Parcel Post, Package Services Media Mail, Package Services Library, Package Services Bound Printed Matter, Express Mail, Express Mail Flat Rate Env., Presorted Standard First-Class, Presorted Standard Class, Global Express Guaranteed, Global Express Guaranteed Non-Documents, Global Express Mail, Global Priority Mail, Global Priority Mail Small Flat Rate, Global Priority Mail Large Flat Rate, International Economy Letter, International Air Letter, International Postcard, International Aerogramme, International Economy Parcel, and International Air Parcel.
    This list has been significantly shortened.
    The new mail classes are: First Class Mail, Priority Mail, Parcel Post, Media Mail, Library Mail, Bound Printed Matter, Express Mail, Presorted First-Class, Presorted Standard Class, International First Class, International Express Mail, International Priority Mail, Global Express Guaranteed, Global Express Guaranteed Non-Documents.

    If you update a template in Vista (likely Windows 7 too, we haven't tested it) save the new file with the same name (ie hit Save, not Save As). If the file has a new name it will not be made available to use in Dazzle. This is because it is saved in a virtual directory in Vista. This is a known Endicia (not Zoovy) bug.

    Mapping Postage Type to Rating/Carrier

    Depending if you are on the latest version of OM, you could have a carrier code field so this may not be needed
    By default Zoovy Order Manager won't know which shipping methods in your store map to which postage types in Endicia. To do this you will need to map the shipping/rating type to an Endicia service. For example you might have a rating method called "Regular Shipping" which maps to "Priority Mail" on the Endicia side of things.

    It is possible to map any shipping method Endicia shipping types. To do this simply prepend a code to the front of the shipping description (the 4 digit code and pipe symbol will NOT be displayed the customers). A list of valid Endicia carrier codes along with their US Postal service names are as follows:
    EFCM|First Class Mail
    EPRI|Priority Mail
    ESPP|Parcel Post
    ESMM|Media Mail
    ESLB|Library Mail
    ESBM|Bound Printed Matter
    EXPR|Express Mail
    EPFC|Presorted, First-Class
    EPFC|Presorted, Standard Class
    EIFC|International First Class
    EIEM|International Express Mail
    EIPM|International Priority Mail
    EGEG|Global Express Guaranteed
    EGGN|Global Express Guaranteed, Non-Documents
    NONE|Do not print postage
    Example: to create a weight based shipping type named "Regular Shipping" and map it to the Endicia Priority Mail type simply enter the shipping description as: EPRI|Regular Shipping (In Web UI, not ZID, new flexible shipping method).
