Verisign/Paypal Payflow Pro

Concept Overview

Zoovy supports the Paypal Payflow Pro gateway (formerly Verisign Payflow Pro). Zoovy does not support Payflow Link.
The management interface is located at https://manager.paypal.com

Configuration

4 Pieces of information are required: Username & Password is the username and password you use to login to the management interface.
Partner: This is typically either "verisign" or "wfb" (Wells Fargo Bank)
Vendor: Vendor is generally set to the same as the Username field.

IP Address Security

Paypal Payflow Pro allows restricting of requests to allowed IP addresses. The IP addresses that Zoovy utilizes for outbound requests are:
