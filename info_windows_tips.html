Windows Networking

Start>>Run Type Control and hit Enter will bring up the control panel

Mapping A Drive

To map a network drive you will need to either know the computer name or IP address of that specific computer or server. Following these steps below will assist you in setting up a Network Drive.
  • Goto Start then Run and type "\\192.168.x.xxx" (Computer\Server IP address) or type "\\computernamehere" (Name of computer or server on the network)
  • A window will pop open of the computer in question with all shared resources on the specific machine.
  • In that explorer window goto Tools>>Map Network Drive>> You will see a window pop Open. You will need to specify a drive letter (Any letter will do as long as you can remember which letter relates to which share) Then you can either select browse or put the address directly into the folder field.
    When selecting browse you will need to navigate through the network down to that folder
    \\192.168.x.xxx\myfolderfromothermachine\database
    Turn on your address bar and you will be able to copy and paste the address into the folder field
