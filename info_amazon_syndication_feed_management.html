Amazon Syndication Feed Management

This addresses how to manage and troubleshoot Amazon product feeds using the Amazon Control Panel in Product Editor

Amazon Control Panel

Each button in the Amazon Control Panel will trigger a specific action.

Save

Will save any changes made to the product's Amazon configuration, and send the current configuration to Amazon.

Validate

Checks a products current configuration against Amazon's required product guidelines. Intuitive errors will be displayed if any of the recommended configuration guidelines are not observed.

Queue Now

Resets the product in Zoovy. Sends the product data to Amazon as if it has never previously been syndicated to Amazon.

Queue Later

Queues product to send 30 hrs after the Queue Later button is clicked. This function can be used in conjunction with the Remove function, for instances when Amazon requires a 24 hr listing removal.

Remove

Removes the product from Amazon. This feature should be used rather than disabling the Allow syndication to Amazon check box (setting amz:ts to 0), if the plan is to re-list the product.
The Save button should be used whenever saving any configuration changes in the Amazon panel. Once the Save button has been clicked, it is generally not necessary to use Queue Now. (Avoiding Queue Now can prevent exhausting the data feed limits Amazon imposes on each merchant account).
Under normal conditions, any product configuration updates should be sent to Amazon automatically.
If the user does not intend to re-syndicate a product in the immediate future, the Allow syndication to Amazon box should be unchecked.
Setting product attribute amz:ts to 0 applies the same behavior as unchecking the Allow syndication to Amazon box.

Feed Details

For additional feed / error details, there are four reference check box options at the bottom of the Amazon Marketplace panel.

Related Items

Displays the feed status of items related to the primary product being referenced in Product Editor.
Note: Although the feed status(es) of the Related Item(s)is displayed for reference, those Related Products' Amazon feeds cannot be updated from the primary product's panel. The Related Product must be searched in Product Editor, thus making it's Amazon configuration accessible.

Summary

Displays a user friendly explanation of the product's feed status, and where the product is in the feed progression.

History

Displays recent events from product's syndication life cycle.

Definitions

Defines terms used in the Status messages.

Amazon Syndication Log

Errors returned by Amazon, which could prevent products from syndicating successfully, can be viewed on a product by product basis in the Amazon Control Panel.
Multiple failed products and their corresponding errors can be viewed in the Amazon Syndication Error Logs.
Syndication >> Amazon Services >> Logs.
