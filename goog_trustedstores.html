Google Trusted Stores

What is Google Trusted Stores

The Trusted Stores Program allows customers to view a seller performance report card and badge on the stores of participating merchants. Merchants who maintain high shipment reliability and high customer customer service ratings will be allowed to display the Google Trusted Stores badge. The program is designed to ensure customers that they are doing business with a reputable and reliable merchant. Google will also act as a liaison between the merchant and customers. Customers can opt-in for up to $1000 in purchase protection. If there is an issue with an order, Google will "work with the merchant and customer to address (it)."

Google Trusted Stores Implementation Guide
Google Trusted Stores set up has 4 steps to complete integration. To begin, login at http://www.google.com/trustedstores

  • - Complete the required business information
  • - Digitally sign the program and shipping agreements
  • - Set up shipping and cancellation data feeds
  • - Grab the code and trusted stores number (upper right-hand corner) for implementation on Zoovy
    Link Trusted Stores to the Google Merchant Center account when completing the shipping and cancellation data feed step. Add required feed file names to data feeds tab. Cancellation filename will be orderstatus.txt and shipping filename is tracking.txt. Under data feeds, click on settings under the cancellation and shipping feed files and set Use Quoted Fields to YES.


    Next, in Zoovy user interface
    Log into Merchant Center and copy the account id # in the upper left hand corner. This is need for the UI
  • - Click on Setup
  • - Click on Analytics and Plugins
  • - Click on Google Trusted Stores
  • - Enter Product Search Account ID, aka Merchant Center ID
  • - Enter Badge Code (below)
  • - Replace variables in code with appropriate Zoovy macros
