Digital Downloads Manager

Concept Overview

The digital downloads manager is an optional module (which can be customized on a per client basis as necessary) that allows users to purchase products which correspond to one or more files that can be downloaded after the sale.

How it works

  • The digital downloads application is installed on a hosting server.
  • The store owner is given an access URL, plus a login and password
  • A supplier (e.g. DOWNLOAD) is created in the store, and pointed at the hosting server.
  • One or more products is associated with the DOWNLOAD supplier in the store.
  • The store owner uploads a file and associates with a PRODUCTID that was associated with the DOWNLOAD supplier.
  • If necessary, multiple files can be associated with a single PRODUCTID
  • When orders are placed (and typically flagged as paid) they are automatically dispatched via supply chain to the DOWNLOAD supplier.
  • The digital downloads module receives the supply chain dispatch and sends one email per file to the client.
