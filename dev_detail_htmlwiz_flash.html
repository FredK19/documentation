Auction Gallery

Paste the following piece of code into your HTML Wizard. You can use the variables listed below to set the color scheme:
Face Changes the background color behind each of the thumbnails and the large image. It also changes the color of the scrollbar and the scroll buttons.
Background No effect at this time. This should still be set, as it may be used in future versions.
Arrows Changes the color of the arrows that appear in the scroll buttons.
Darkshadow Changes the color of the outline around the thumbnail and the large image window pane. It also changes the shadow color on the scroll bars, pane, and buttons (right and bottom strokes).
Highlight Changes the color of the inside highlights in the scroll bars, pane and buttons (the inner-stroke on the top and left).
Highlight3d Changes the color of the outside highlights in the scroll bars, pane and buttons (the outer-stroke on the top and left).
Scrolltrack Changes the color of the scroll pane.
Selection Changes the stroke and background color of the thumbnail that is currently selected.
Textselected No effect at this time.
Textcolor Changes the color of the product names that appear below each thumbnail.
Textsize Changes the size of the product name font.
At this time, there is no way to change the font face or font color of the details that appear below the gallery images.
