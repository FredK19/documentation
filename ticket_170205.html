XML Parsing Error: 0 Where to find Error.xml File

Issue: XML Parsing Error(error 0)

For Zoovy techs to resolve this issue, you'll need to enable Sync Logging, and attach the resulting sync log file to a support ticket for the techs to trouble shoot.
To enable Enhanced Sync Logging
  • Open ZOM and click Admin > Technical Support > Log Sync Files (A check mark indicates this feature is enabled)
  • Run a sync to reproduce the error
  • From the Windows Start buton, navigate to c:/program files/zoovy/desktop/logfiles
  • Zip up the files in this directory into a single zip file and attach to the ticket
    • Hint: Consider searching for 'warehouse manager error 0'at ttp://webdoc.zoovy.com
    NOTE: This also may display in ZID version 7.032 and higher as a "2025 error"
    VISTA USERS
    If you are using Vista and your UAC (User Access Control) is turned on, the directory you are going to look in is:
    c:\users\(username)\Appdata\local\Virtual Store\program files\Zoovy\Desktop
    Note: You may need to make hidden folders and files visible. This can be toggled through Folder Options
