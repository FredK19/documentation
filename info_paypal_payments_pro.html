Overview: PayPal Website Payments Pro

Zoovy is fully certified for Paypal Website Payments Pro.
Setup Instructions
  • Login to Zoovy.com
  • Click Setup | Payments
  • Select the Credit Card tab
  • Select Website Payments Pro

    • Usage

    Refer to the Paypal Express Checkout (which shares settings with Paypal Website Payments Pro) for more configuration information.
