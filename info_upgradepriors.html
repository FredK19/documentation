Prior to Your Upgrade from Stand Alone to Network Appointments

Concept Overview

Things to do prior to your upgrade appointment

Before your upgrade to Network appointment

Once you have setup your appointment for upgrading to Network order manager go to the download section in your Zoovy acount (this is located at the top right of your screen) and you will find links, then you will need to do the following:

On the master machine: Download and install the following:
  • .Net 3.5 Framework sp1 (will require rebooting the computer).
  • MySQL Connector/Net 5.2.5 - Windows Binaries (zipped MSI installer)(should be around 3.3-3.9MB file)
  • Windows Installer 4.5 (may require rebooting the computer).
    Windows Installer 4.5
    If you are running Windows XP select WindowsXP-KB942288-v3-x86.exe. If you are running Windows Vista select Windows6.0-KB942288-v2-x86.msu

    Download but do not install:
  • MySQL 5.1 - Windows MSI Installer (should be around 103-105MB file
  • Zoovy Sync Manager
    To download them without installing, when you click on the download link a box should pop up and give you the choice to run or save. Choose save and save it to the desktop.

    Placing these items on the desktop is usually the easiest place to find them and allows for a quick check by Zoovy techs to make sure all the files are there and ready to be installed.

    • For each client

    On each machine you want to use Order Manager / Warehouse Manager: Download and install the following:
  • .Net 3.5 Framework sp1 (reboot the computer).
  • Windows Installer 4.5 (may require rebooting the computer).
  • MySQL Connector/Net 5.2.5 - Windows Binaries (zipped MSI installer)(should be around 3.3-3.9MB file)
    Download but do not install:
  • Zoovy Integrated Desktop
