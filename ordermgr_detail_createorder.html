Create an Invoice

Overview

With the Zoovy Order Manager you can easily create invoices from incomplete sales made on portals like eBay or Yahoo Auctions. The functionality of the invoice create feature mirrors that found on the website manager. This feature stemmed from merchants who needed to easily turn incomplete sales to valid orders when their customers decided not to use Zoovy's checkout system.

Invoice Creation

Accessing the Creating Invoice tool is done from several different areas. Clicking the Incomplete tab will enable the Invoice icon found on the top of the screen. Click this icon to begin the process. You may also right click an incomplete sale and select Create Invoice from the pop-up box. Lastly, the Create Invoice tool can be accessed from the incomplete order search tool. When an incomplete order is found, simply click the invoice icon.

    Bill To Tab

  • Email: often this field is filled in when creating orders from portals sites like eBay. This field can be modified if necessary.
  • Full Name: enter the customers full name here. The name entered here will be used on invoices and the like.
  • Smart Address: the Smart Address field allows for quick entry of address information. Payment notifications from services like PayPal can be easily cut-and-pasted into this field. Street address, city, state, and zip fields are automatically pulled out of a single text box. It is important to follow the format specified to the right of this text box.
  • Sales Tax Rate: if tax should be charged for this order, enter the correct percentage here.
  • Shipping Amount: The shipping charges for the sale should be entered into this text box. If you wish to specify a specific carrier, or shipping method such as "Default Shipping" type your desired entry into the via text box.
  • Notes: add an internal note to the created invoice by entering text into the Notes text box.
  • Set New Order as Paid in Full: Check this box to create the order invoice as paid. You will not have to later change the invoice to paid.

    • Bill Info Tab

    Payment Type: this drop down box can be used to specify which type of payment a customer will be using. It may be left blank when payment type is not necessary to specify.

    Finishing Up

    To complete the process simply click the SAVE icon located on the top left of the screen. The created order will be placed into the pending orders tab. Alternatively click the cancel icon to cancel the process and close the screen.
