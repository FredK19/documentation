Issue #158588: Quickbooks Troubleshooting Information

In order to diagnose a Quickbooks issue - the development team will need the following information:
  1. Attach Screenshots for all three QuickBooks tabs in Order Manager, including product creation.
  2. The qbsdklog.txt typically located C:\Progam Files\Intuit\Quickbooks Pro . If you cannot find it, you will need search your hard drive.
    Vista Users: You need to use the advanced search functionality in order to find the qbsdklog.txt . To find the file, click on search from the Windows button in bottom left hand corner. Then click on Advanced Search in the upper right hand corner of the Search window. Then in Location change the drop down from Indexed Locations to Everywhere. Also you need to check the following check box: Include non-indexed, hidden, and system files (might be slow). In the name box type qbsdklog.txt and hit the search button.
  3. The Item list file inside QuickBooks. To get this you need to do the following:
  4. login in single user mode and admin user inside Quickbooks
  5. In menu area of Quickbooks, File -> Export -> Lists to IIF files.
  6. Check the Item list and hit "OK"
  7. The version of Quickbooks you are running (examples Quickbooks Pro 2005, Quickbooks Premier 2006, etc)
  8. The exact error that you are getting when trying to export, if any.
  9. The order numbers from Zoovy that you are trying to export.
  10. Please also provide the order number in question.
