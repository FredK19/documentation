Configuring Promotions with Checkout By Amazon payments

Promotions with Checkout By Amazon

If you set up a promotion or coupon in Zoovy and want it to be available to Checkout by Amazon (CbA) customers, you need to configure it within Seller Central, as well as in your Zoovy settings.
To create a promotion in CbA
  • Log into your account at sellercentral.com
  • Select, Your Checkout Website from the dropdown at the top (if your acct is separate from your Seller Central acct, skip this step)
  • Click the Settings Tab
  • Click Manage Promotions
  • Create a new promotion
  • Enter the information for the promotion.
  • Type - Select the type of promotion you want to use. Note that while you can have multiple promotions and you can combine promotions (such as free shipping + 10% off), an individual promotion must be just one type.
  • Start and end dates - Select the start and end dates for the promotion. Note that the time is Pacific time, whether Standard or Daylight. Click the calendar icon to pick a date from a calendar, or enter your own.
  • Tracking ID - Enter a description for the promotion for your order and settlement reports. This description does not appear to your customers.
  • Internal Description - Enter a description for your own use in the Manage Promotions feature. This description does not appear to your customers.

  • Click the Continue button.
  • Select whether to use claim code options.
  • Fill out appropriate info.
  • Click Continue.
  • Fill out needed info for Defining Qualifications
  • Click Continue
  • Select eligibility of destination/shipping methods
  • Click continue
  • Review and submit promotion.

    • If you have any issues with promo creation, please schedule an appointment with implementation. For more information on Amazon and Promotions, please see this document.
