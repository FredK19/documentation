Wrapper Reference - Categories

Overview

In the codes below, there are several instances of ATTRIBUTE=0 or 1. In these cases, 0 is false and 1 is true. So in the instance of NOHOME=1, the home button/text would not appear in your navigation categories. Pay close attention to the image categories, which sometimes duplicate a true or false statement up to three times. Make sure all three are the same, or funky things will happen with your categories. In image categories, feel free to adjust the width of the buttons, but again, do it in all three places. BUTTONTYPE=sleekgrey4_off can be changed to any button type in the zoovy system. It would need to be changed once in the body code and twice in the head code. All three references can point to different navbuttons.

Your code does not need to contain the nohome and onlyhome lines of code if you are not using them. By default, both are set to zero. In some cases (text navcats), there is a divider line of code. It be used to set what is placed between two categories. Text menus also offer the 'after' tag, which can be used to place a piece of code on a page ONLY if there is navigation that appears. Some notes about subcategories:
NOROOT=1 This will make sure the root categories do not appear in the subcat area. Set this to 0 and navigation categories with no subcategories will display the rootlevel navcats in the subcat area.
DESCENDTREE=1 This will display the latest (or lowest) level of subcats displayed if no new subcategories are available.
PRODCAT=1 This will show all the categories and subcategories on a product page that the product resides in. Right now, this includes hidden categories.

Horizontal Text Navigation Categories

Place this code into your ZHTML where you want your root-level navigation categories to appear:
<!-- MENU FORMAT=<a href="%URL%" target="%TARGET%">%MESSAGE%</a>%BREAK% DIVIDER=|%BREAK% AFTER= ONLYHOME=0 NOHOME=1 -->

Horizontal Image Navigation Categories

The image based navigation needs two pieces of code to work properly. The piece immediately following should be placed in your ZHTML where you want the Navcats to appear.
<!-- MENU BUTTONTYPE=sleekgrey4_off FORMAT=<a href="%URL%" OnMouseOver="on(%NUM%)" OnMouseOut="off(%NUM%)"><img src="%SRC%" width="%WIDTH%" height="%HEIGHT%" border="0" name="roll_%NUM%"></a>%SP% WIDTH=140 NOHOME=0 ONLYHOME=0 -->
Place this code in the head of your ZHTML:
<script language="JavaScript"> <!-- initialized = 0; <!-- MENU FORMAT=rollover_%NUM%_on = new Image;%BREAK%rollover_%NUM%_on.src = "%SRC%";%BREAK% BUTTONTYPE=sleekgrey4_on WIDTH=140 NOHOME=0 ONLYHOME=0 --><!-- MENU FORMAT=rollover_%NUM%_off = new Image;%BREAK%rollover_%NUM%_off.src = "%SRC%";%BREAK% BUTTONTYPE=sleekgrey4_off WIDTH=140 NOHOME=0 ONLYHOME=0 -->initialized = 1; function on(imageNumber) { if (!initialized || !document.images) return; document.images[eval("\"roll_"+imageNumber+"\"")].src = eval("rollover_"+imageNumber+"_on.src"); } function off(imageNumber) { if (!initialized || !document.images) return; document.images[eval("\"roll_"+imageNumber+"\"")].src = eval("rollover_"+imageNumber+"_off.src"); } // --> </script>

Thumbnail Categories

If you want to use the category thumbnails in your layout, you'll need to use different code to display the categories. This code is faster than the standard menu wrapper element, however does NOT allow you to use the graphical buttons features. Paste the following code into your wrapper. Leave source set to . if you want to display the root level categories. Alternatively, you could set this to SRC=.safename (where safe name is a category safe name) and it will pull in a set of subcategories. This is handy if you want to break your navigation into separate sections. Use the variables on the category flow elements for this.
<!-- ELEMENT TYPE=SUBCAT MODE=LIST SRC=. FORCECOLUMNS=1 <HTML>= &lt;!-- ROW --&gt; <tr> &lt;!-- CATEGORY --&gt; <td><a href="%cat_url%" ><img src="%image_url%/W40-H40-Bffffff/%cat_thumb%.jpg" height=40 width=40 alt="%cat_pretty%" border="0"></td> &lt;!-- /CATEGORY --&gt; </tr> &lt;!-- /ROW --&gt; </HTML>= -->
