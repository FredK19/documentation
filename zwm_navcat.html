ZWM Categories

Concept Overview

The Warehouse Manager Desktop Client allows customers to easily add/remove products from navigation categories and lists that are used on their website.

Building Navigation in ZWM

You can build your website navigation in Warehouse Manager. Select a webstore category under "Your Website" and right-click on it. Select "Add Website Category." An interface will appear that will allow you to input the category name. Give it a name and click on "OK."
