Configuring Outlook Express

 

 


Setting up your email Client for FuseMail. - Outlook Express using IMAP

This tutorial explains how to set up Outlook Express for use with FuseMail.


Before you start
You will require some details from when you set up your FuseMail account.
You require:
The IMAP Server Name - imap.fusemail.net
 The SMTP Server Name - smtp.fusemail.net
 Your FuseMail Username - (for the purposes of this tutorial we will use demo)
Your FuseMail Password - (for the purposes of this tutorial we will use demo)

Open Outlook Express and from the top menus select Tools then Accounts...


Select Add > Mail...

The name you type in here will be the name emails appear to come from

Type in your name (in this tutorial we will use John Smith) and click Next>.

The email address you type in here will be the address people use to send you emails.

Type in your email address (in this tutorial we will use demo@fusemail.com) and click Next>.

The server names. This tells outlook express where your emails are stored.

Select IMAP from the dropdown field.
Type in the IMAP server in the Incoming mail field (imap.fusemail.net)
Type in the SMTP server in the Outgoing mail field (smtp.fusemail.net)
 Click Next>

Internet Mail Logon.

In the Account name: field type in your fusemail user name (for zoovymail customer, use the ENTIRE email address, ie, admin@USERNAME.zoovy.com, where username is YOUR account name).
In the Password: field type in your fusemail password (in this tutorial we will use demo).
Click Next>

Finish.

Click Finish to complete the wizard.

Mail Accounts. You will see your new FuseMail account in the list.

Click on it and then Click Properties to continue.

Click in the Servers Tab.

Check the check box and then Click Settings to continue.

This will display the following dialog.

Click OK to continue.

Click on the IMAP tab

 To store your sent emails and drafts on FuseMail, rather than on your computer you need to change a setting in the mail account properties.


Type INBOX in the Root Folder Path field

Click OK to continue.

Mail Accounts. You will see your new FuseMail account in the list.

Click Close to continue.

Click YES to continue.


Wait for the folders to be syncronised.



Click OK to continue.


 You will notice that you now have your fusemail account in your folder list within outlook express

 

You can now use Fusemail with Outlook Express.
