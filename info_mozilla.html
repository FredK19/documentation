Copy & Paste in Mozilla Firefox

Unable to copy and paste in Mozilla Firefox browser


When using the Mozilla Firefox browser. I am unable to right click and copy/paste. I receive the message "Unpriviledged scripts cannot access cut/copy/paste programatically for security reason." I can copy/paste on other sites. I have no problem when using Internet Explorer
This is a common issue with Mozilla Browsers and is not unique to Zoovy. Often using the keyboard shortcuts will bypass this issue. Ctrl C copies. Ctrl V pastes.
Always make sure you have the latest official updates for your browsers. Avoid using beta releases of browsers.
