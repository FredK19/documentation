Issue

Issue: errors when creating an order in OM
I have a fresh install of ZID on a windows vista machine. when i click the create order button in order manager, i recieve the error ACCESS TO THE PATH C:/PROGRAMFILES/ZOOVY/DESKTOP/ORDERIDS.INI DENIED See the end of this message for details on invoking just-in-time (JIT) debugging instead of this dialog box.
************** Exception Text **************
System.UnauthorizedAccessException: Access to the path 'C:\Program Files\Zoovy\Desktop\OrderIDs.ini' is denied.

Resolution

When opening ZID, click on admin (before logging in) > account security You should be able to update after verification.

Resolution


Some tracking down found this to be an issue with Windows Vista.
What you can do is copy the file ORDERIDS.INI from the following directory: C:\Program Files\Zoovy\Desktop
and place it in the following:
c:\Users\(user)\Appdata\local\VirtualStore\program files\zoovy\desktop\(Zoovy account name)
::Where parentheses stands your computer's personal folder names and usernames. It may be easier to manually search through folders. I noticed the folder "Appdata" is hidden, so you may need to enable view hidden files to see this one.
If you have any further inquiries, or need further assistance with this please let us know.
