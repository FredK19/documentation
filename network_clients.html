NETWORK CLIENT SETUP

Concept Overview

If you are not unsure what you are doing you might want to consider scheduling with our GEEK SERVICES to do this for you.

For each client

On each machine you want to use Order Manager / Warehouse Manager: Download and install the following:
  • .Net 3.5 Framework sp1 (reboot the computer).
  • Windows Installer 4.5 (may require rebooting the computer).
  • MySQL Connector/Net 5.2.5 (should be around 3.3-3.9MB file)
  • ZID (Order Manager) 8.xx
    If you have a network folder, a folder that is shared across all computers on your network, to save time on downloading you should save these files to the network folder. They can be install from the network folder to the computers that can reach the folder.

    • Set up of Network Client

    You need to have a network or VPN set up in order for this to work. (i.e. your computers need to talk to each other)
    After you have successfully install the software and components including ZID follow the next step to connect to your server:
  • Open ZID by double clicking on the icon or right clicking on the icon a selecting open
  • In the top left select Admin (A pull down menu will appear)
  • Select Database Config and click on it
  • A Window will open with a tab that says 'Standalone' and a tab that says 'Network'
  • Select the Network tab
  • Now enter your Username and the IP address of your server
    You need to get your IP address from your server in the cmd prompt with ipconfig
  • Now click save and log into ZID.
    If you are getting an error on a client but not on the server, you should check to make sure that all of the software is installed. You can check this in 'add/remove programs' in Control Panel. Also check if there have been any changes to the Managed Users section of your Zoovy account. A nuke config may be needed if there has. Contact Zoovy support if you need assistance. Check port 3306 on server.
