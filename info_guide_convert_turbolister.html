Converting from Turbo Lister

Overview

Turbo Lister is an auction launching (not closing) tool provided by eBay.com - it competes directly with Sellers Assistant Pro.

It is possible to export Turbo Lister listings into a CSV file for import directly into Zoovy - for more information: http://pages.ebay.com/turbo_lister/csv.html

Importing Data

A Turbo Lister import is done by simply importing the running eBay auctions. Please read the details on importing listings from eBay.com Click Here
