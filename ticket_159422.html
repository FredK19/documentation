Issue #159422: Customer can't see their order in the customer portal / order history

Issue

The order history in the customer portal is what a client see's when they login to a hosted store. It displays order history so customers can track their order history. In order for this to work, customers must have an account on the website.
Merchant received this:
Hello, my order history is not operational; this order is not showing.

Resolution

The customer either created their account after placing the order and/or used a different email address to create their account - so the order is not associated with the account.
If you are experiencing this issue, check for the solutions above - then please be sure to include the order # in the ticket.
