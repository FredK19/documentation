Batch Channel Wizard

Batch Channel Wizard Overview

The Batch Channel Launcher is used to create multiple EBay or OverStock listings at once.

How to use the Batch Channel Launcher

The Batch Channel Launcher is found under the Products tab, in the left column.
It is a good idea to be comfortable with launching channels individually before you attempt to use the Batch Channel Launcher.

Usage

Batch channel launcher is intended to launch large batches of auctions at one time. To make the most of this tool, you should consider using the product power tool to set your HTML auction templates.
'Steps'
  • Select your destination marketplace
  • Set your scheduled launch time. If you do not set this value, the auctions will launch immediately.
  • Select your products either individually, or by category. If you have many products, it could take a while for the individual selection page to load
  • Select the fields you will be customizing data for.
  • Hit enter data. Each field you have selected in the previous step will appear here for data entry or review. You must hit refresh on the HTML tools.
  • Now select launch channels

    • Select Attributes

    The Select Attributes step is sometimes a little difficult to understand. In this step, you are designating which attributes that you will change with this batch launch. If you have already launched the products that you are selecting and simply want to launch them again in the exact same way you would select only a few attributes to ensure that the values are correct. If you have never launched the products that you have selected you will want to select most attributes to ensure that you have set them successfully.

    Saving to Product Database

    This will save time when you launch the next batch of channels. All of the data you have chosen to enter as attributes will become part of the product definition.

    Summarize

    It is a good idea to click on the Summarize button before you do a batch launch. This will give you an overview of the attributes that you have just set and allow you to catch any mistakes before the auctions are live on EBay. Only after you are certain that you have built your listings correctly should you click on the "Launch Channels" button.
