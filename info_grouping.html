Grouping

Grouping Overview

Merchandising can be difficult when dealing with several different marketplace. Grouping provides some solutions to merchandising complications. Grouping will allow individual option choices to appears in searches. For example, if a customer searches for "red bikini" they can find "red bikini" rather than a generic bikini that may have an option to select "red" but the customer will never know without first clicking on it. Grouping allows easier syndication into eBay for merchants who use options. Also, grouping allows you to display multiple options on Amazon, limiting the option choice to which color and size are available and in stock.
Grouping is one of many ways of handling options. It is not the only way.

Fields

Fields Assigned to the Parent

  • zoovy:grp_type - This should be set to PARENT for the parent product. It defines the role of the product. If you set zoovy:grp_parent first it should automatically set this field on the product.
  • zoovy:grp_children - You populate this field with the Product IDs of the children, separate by commas.

    • Fields Assigned to the Child

  • zoovy:grp_type - This should be set to CHILD for the child products. It defines the role of the product. If you set zoovy:grp_parent first it should automatically set this field on the product.
  • zoovy:grp_parent - You populate this field with the Product ID of the parent.
    • By populating the field zoovy:grp_parent on the children, you can automatically create the parent Product ID with all appropriate fields set for grouping.

    Amazon Fields

  • amz:grp_varkey - This is the variation keyword that represents the group on Amazon. For example, "APPAREL: Color". This field should be set on both the parent and children.
  • amz:grp_varvalue - This is the name of this option when displayed on Amazon. For example, you would not want to display the entire product name when the "Black Suede Boots" are on Amazon. You would simply show the choice "Black". This field should be set on the child products.
