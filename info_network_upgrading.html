Upgrading NETWORK Order Manager

Overview

If you will be using the NETWORK Order Manager, this document will cover the steps you need to take.
Unless otherwise informed by Zoovy Technical Support, you MUST upgrade SyncManager FIRST before upgrading the client.

Click by Click Upgrade
  • Step 1 - Uninstall SyncMgr on Database Server Machine, Reboot the machine
  • Step 2 - Install new SyncMgr version on Database Server Machine
  • Step 3 - Sync
  • Step 4 - Uninstall ZOM Client on client machine, Reboot the machine
  • Step 5 - Install new ZOM 8.x Client on client machine

    • While the "reboot the machine" action above isn't strictly mandatory for the upgrade to work, it is a generally acceptable procedure and a good practice to insure trouble free upgrades.

    dot net framework error

    When upgrading from version 7.056 to version 8.0x or higher you will encounter the following error when trying to install and connect an Enterprise Order Manager, and Sync Manager client to the mysql database for the first time: ************** Exception Text **************
    System.ArgumentException: Unable to find the requested .Net Framework Data Provider. It may not be installed.
    at System.Data.Common.DbProviderFactories.GetFactory(String providerInvariantName) at Zoovy.Common.Auth.ZAuth.OpenCnnDB()
    at Zoovy.Common.Auth.frmLogin.DBLogin()
    ......(con't)

    The error is resolved by downloading and installing a connection component from MySql. The component can be found by following the link below:
    mysql-connector-net-5.2.5.exe. Choose the Windows Binaries Download.
    Simple click the link and then click on "No thanks, just take me to the downloads!". Choose any mirror site and "HTTP". Running this on your machine will resolve the connection issue and allow EZOM client to reach the database.
    You must uninstall any previous version of the connector that is not 5.2.5 before you download the new one. If you do not then it will not allow you to download it.
