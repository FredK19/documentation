Product Power Tool Tips

USERS BEWARE: This tool can make mass changes to all your product in an instant.
If you do not use this tool properly it could corrupt your product database or make undesireable changes to yoru products.
Please seek training prior to using this tool. Fixing any problems caused by this tool may be billable by support

Concept Overview

This doc is dedicated to giving you helpful tips when using the product power tool.

Changing Products En Masse
  • 1. Go to Utilities > Product Power Tool.
  • 2. Select the products that you wish to change.
  • 3. From the "Select Attribute" dropdown menu, choose the attribute you would like to change. If you want to change an attribute that is not in the drop down list then select "Custom Attrib."
  • 4. If you chose "Custom Attribute" then type in that attribute in the "Custom Attrib" box.
  • 5. Make the choice in Select Action interface that best fits the change you want to make.
  • 6. For Select Update Type choose "Products."
  • 7. It is usually a good idea to leave "Preview before making changes" checked. It will help you avoid making unwanted mistakes.
  • 8. Click Save and the Preview screen should appear.
  • 9. If it looks like you are making the correct change, click "OKAY."

    • Deleting Product Attributes
  • 1. Go to Utilities > Product Power Tool
  • 2. Select the products that you wish to remove a product attribute from
  • 3. For "Select Attribute" choose the attribute you would like to remove. If you want to remove an attribute that is not in the drop down list then select "Custom Attribute"
  • 4. If you chose "Custom Attribute" then type in that attribute in the "Custom Attrib" box
  • 5. For Select Action choose "Delete Attribute from product record"
  • 6. For Select Update Type choose products
  • 7. Then uncheck "Preview before making changes" (If you do not uncheck this, it will not delete the field)
  • 8. Click Save and it will remove the attribute from all the products that you selected
