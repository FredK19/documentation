Shipper File In ZOM

Creating Shipper Import File In Order Manager

Zoovy Order Manager supports the importing of order information using the "Shipper Import File". This creates venue to where tracking for orders can be imported from external sources that are not supported directly by Zoovy.

Creating an Import Shipper File

You need to create a file in CSV format. The order of the fields is critical when preparing this file. Here are the available fields:
* tracking # - example: 123456789 * order # - example: 2008-01-12345 * method - example options: UPS, USPS, FEDEX * cost * actual wt * notes
Note that you don't need any headers, just the fields in the correct order.
You do not have to use all the columns, but you will want to leave spacers (double comma)if you are leaving a middle field empty to hold the place. Example - if I only want tracking and order number (required), my file would be:
123456789,2008-01-12345 987654321,2008-01-12346 ...etc.
If I wanted to not put in the method, but put in cost it would be:
123456789,2008-01-12345,,5.67 <<--- NOTE THE DOUBLE COMMA 987654321,2008-01-12346,,3.92 ...etc.
