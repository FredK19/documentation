Finder

Finder Overview

A Finder is a predefined set of search parameters which will allow your customers to easily find what they're looking for. For instance, a dropdown list of 'genres' to help your customers find what toy they're looking for. Or checkboxes of 'colors' to allow them to find clothing of a specific color.
Finders can also contain multiple search parameters and types, such as clothing style as a dropdown and color as checkboxes. That way the customer can search for hats that are red, blue or khaki.
Regardless of whether you decide to use the theme, or layout enabled finder configuration, you will need to configure a FINDER search catalog in the Setup >> Manage Search Catalogs interface.
A 'product finder' is different than a finder. A product finder is used in the Zoovy interface to allow merchants to find and select product to appear in a given list or category page.

Adding a finder to your store

Using the right theme or layout

In order to use a finder, you must be using a theme or layout that supports one. Shoptastic and Bluelicious are two examples of themes that have a finder. There is also an advanced search layout that has multiple finders (ID: 5006).

Mapping your products

In order for a finder to work, you must first create an option group to be searched. Option groups are covered more thoroughly in this document: Doc 50319
Option groups must be Store Option Groups (SOGs), as Product Option Groups (POGs) will not be recognized by the search engine.

Once the option group has been created, you must map your products to the appropriate data fields.
Mapping your product to an option group 'en masse'
  • Click on the utilities tab
  • In the Account Utilities section, click the link for Option Power Tool
  • Select all or some of your products
  • Choose the option group you wish to associate product with and push the edit button
  • Check the appropriate boxes for each product
  • Save your changes

    • Enabling the finder via layout/theme

    Now that you have your product mapped to specific option fields, you need to set the finder up to search this option group. Edit the page/theme that has the finder element in it.
    Enabling the Finder
  • Click 'edit' next to the finder prompt
  • Enter a prompt that will appear on your store next to the search field (this should be short and reflect the option group that will be searched)
  • Select the group to be searched
  • Check/Uncheck 'ANY' as needed
  • Select the display (most DEFAULTS are a select list)
  • Push the 'Add to Finder Text' button
  • If you are adding multiple search parameters, repeat these steps. If there will only be one search parameter (one dropdown, for instance)then you are done. Click the save button
  • Check the appearance of the finder on the theme/page you are using it to ensure aesthetics
    • A finder may be template specific, so check the theme or layout you are using in the editor for special instructions

    Configuring a finder in the Search Catalog

    Configuring a finder in the Search Catalog
  • Click Setup >> Click Manage Search Catalogs
  • From the Create a New Catalog section, name the Catalog Code:FINDER
  • For Format, select Finder Product Map from the drop down
  • The new catalog now appears in your list of search catalogs. Clicking Refresh will update your product records.
    • Note: These record updates can take a long time depending on the size of your product data base and how many products are being updated by a specific refresh
    A search catalog configured as a Finder must be named FINDER for it to function correctly
