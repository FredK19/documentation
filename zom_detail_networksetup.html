Order Manager - NETWORK

NETWORK Overview

With the NETWORK bundle, the Zoovy Order Manager can be installed on multiple computers and all stations may utilize a common database. There are two ways to get multiple user support: Multi-User Network, and Enterprise.
NETWORK allows the Order Manager database to be installed on a shared network drive and then be accessed by multiple installations of the software. A single workstation is designated as the primary station, the primary station is the one which is allowed to synchronize and make changes to the company configuration. Additional stations can be used by customer service personnel, but those personnel should not synchronize.

Considerations

The station with sync manager is the only station which that needs to sync, the client machines do not need to sync as they will read the database from the syncing machine. At this time you cannot switch which station is considered the primary (we plan to support this in a future build). If you are operating the primary station - you should not synchronize while people are logged in to the database, doing so could cause corruption.

NETWORK config

There are different configuration steps depending on your Operating System. Please follow accordingly.
ALL MACHINES ARE REQUIRED TO HAVE THE SAME VERSION OF ZID RUNNING

Windows XP for ZID

Windows XP Master machine

  • Run ZID. You will be prompted to configure the Database. Log in the NETWORK tab using your Zoovy login information. Once done, at the log in screen, click admin > account security. Log in using your Zoovy log in information. Exit ZID
  • Log into ZID
    • Windows XP Client Machine for ZID
  • Install ZID to default directory. Be sure it is the same version as the Master
  • Run ZID. When Prompted to set up Database Config, click on NETWORK. You will be asked for the Zoovy store (login) name. There will be a field for the IP Address of the server. Fill in your servers IP Address.
  • Click "Save"
