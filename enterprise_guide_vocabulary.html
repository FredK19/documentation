A

Active - The state assigned to a channel when it is live and running on a marketplace. In most circumstances Active listings cannot be edited.

B

C

Channel - a way to transmit information to a marketplace. In laymens terms: What you create to launch an auction listing.

D

Dispatched - the term dispatched means that the channel has been received by zoovy and sent to the marketplace, or more specifically the system which interfaces Zoovy to the marketplace. Typically each marketplace has a TODO queue which is the list of work it will do imminently.

E

"E" (Error) Channel State - A channel has experienced an error, either while running or during launch. If the listing has not launched then you should fix the error and reset the channel to P/Pending (the system will do this automatically if you edit an Error channel).

M

Marketplace - technically any place that can have a channel launched to it e.g. eBay.

Marketplace ID - the internal listing number used by the marketplace to track the listing. e.g. eBay Item #.

O

P

"P" (Pending) Channel State - a state used for channels which have been transmitted to Zoovy and have been allocated a channel number, but have not yet been dispatched to the marketplace.

Q

R

"R" (Revoke) Channel State - revoking a channel tells Zoovy that it should not relist the listing. This means you are safe to close the listing via the marketplaces user interface (e.g. go to eBay.com and cancel the listing).

S

"S" (Schedule) Channel State - a state used to describe channels which were created with a channel schedule record. This means the channel has been been transmitted to Zoovy, along with a valid scheduler record and will launch one or more times. If a channel launches more than one time, it will clone itself and make a copy.

Wrapper - the html file which defines the navigation (e.g. categories, along with the header and footer) for a site hosted by Zoovy.

Z

Zombie - term used to describe when a channel has been revoked but has not exited. A channel state of "Z" is sometimes assigned to zombie channels.
