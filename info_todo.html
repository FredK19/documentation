TODO List

Concept Overview

The Zoovy To Do "TODO" list tracks outstanding tasks that you need to complete such as correcting errors, or responding to support tickets. Upon login if there are any tasks in your todo list then the "TODO" button in the upper right hand corner will flash.

Add a Custom Task

By clicking on the "MORE" link to the far right, you can add your own Tasks to your To Do list.

Actions

Set as Completed

When you have accomplished a task, set it as completed.

Remove from List

If you find that a task is no longer necessary you may remove it from your To Do list without completing it.

What else should I do?

Be sure to schedule an Implementation Appointment so that one of our Zoovy technicians can assist you with getting through the To-Do List. To schedule an appointment, go to Support > Request Implementation Assistance.
