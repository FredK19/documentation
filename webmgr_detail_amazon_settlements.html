New Amazon Shoe Templates

Overview

Amazon's Shoes, Handbags, Sunglasses/Eyewear category has been split into 4 separate templates:

Shoes
Shoe Accessories
Eyewear
Handbags

Using the new templates:

Please follow the steps below to correctly configure NEW (not yet syndicated) products to syndicate using the new templates. These same steps can also be followed to modify any currently syndicated products (recommended). Products syndicated using the old Shoes, Handbags, Sunglasses/Eyewear templates prior to Amazon's deadline (5/31/12) will continue to syndicate without issue. However, we recommend updating your currently syndicated products to the new templates as well, in order to optimally position them on Amazon, and make it easier for shoppers to find them.
1. In Product Editor, scroll to the Amazon Marketplace panel, and from the "Amazon Product Type" dropdown menu, select "Shoes", "Handbags", "Eyewear" or "Shoe Accessories".
2. Click Save.
3. Click on "Enter Specifics"
4. Copy /Paste the following url into your web browser to view Amazon's BTG for Shoes, Handbags, Sunglasses/Eyewear:
https://images-na.ssl-images-amazon.com/images/G/01/rainier/help/btg/shoes_browse_tree_guide._TTH_.xls
5. BTG displays 3 sheets. Open the sheet titled "SHOES (5.10.2012)".
6. Column B lists all of the Shoes, Handbags, Eyewear and Shoe Accessories categories. Select the appropriate category.
7. After choosing a category from column B, find the corresponding "department_name" in Column C, and select that value from the "Department" dropdown menu in Zoovy.
8. Next, find the corresponding "item_type_keyword" value in Column C and copy /paste it into the "Item Type" field in Zoovy.
9. Click Save.
For those who are comfortable using the csv product upload tool, the "Amazon Product Type", "Department" and "Item Type" can be added in bulk to your products using the attributes (headers) "amz:catalog", "amz:prod_cloth_cd_itemtype" and "amz:prod_cloth_cd_dpt".
10. Complete any other fields that are relevant to the product you are selling.
Amazon suggests additional attributes to use for each category via their Inventory File Template spreadsheets. Copy /paste the following url into your web browser to view Amazon's Inventory File Template for Shoes, Handbags, Sunglasses/Eyewear:
https://images-na.ssl-images-amazon.com/images/G/01/rainier/help/ff/release_1_9/upgraded_templat es/Flat.File.Shoes._TTH_.xls
11. Click Save.
