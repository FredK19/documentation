Customer Manager

Overview

You can create a single customer by going to Utilities > Customer Manager and clicking the "Create" button. Using the Zoovy Order Manager for Windows you can add customers by going into the Customer Tool and clicking the "Create" icon. You can import large lists of customers using the Import tool which can be found in Setup | Import Tools. If you are a programmer you can also synchronize customer databases with other remote databases using the Zoovy WebAPI which can be found at http://www.zoovy.com/webapi

Editing Customers

You can edit a customer by going to Manage | Customers and then searching for the customer to edit.

Exporting Customers

You can export a list of customers email addresses by going to Utilities > Customer Manager and clicking on the "List All Customers" link (found in the blue bar on the page). This will create a report of all your customers name and email addresses. As this point you can choose to export into an MSExcel by clicking on the MSExcel "X" icon in the upper right corner of the report.
Please note that the best way to send mass email is using the Zoovy Order Manager, in this case Zoovy will automatically generate the opt-out link at the bottom of your messages and track it with their account preferences. If you use a 3rd party you must make sure they comply with Zoovy's very strict anti-spam rules or you run the risk of having your site suspended.
