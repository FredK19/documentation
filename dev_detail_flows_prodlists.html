Product Lists

Overview

Overview: Product lists can be used on product, category, cart, search results or homepage flows. How the products are displayed in a product list is known as the 'listing style'. Some of the default Zoovy listing styles include: thumb, plain and detail.

Default Product List:

<ELEMENT TYPE="PRODLIST" NAME="PRODLIST1"> PRETTY=Product Listing ABOUT=You can add products to your homepage, when you do they will appear below. MULTIPAGE=1 DEFAULT=THUMB,1,2,NONE </ELEMENT>

Default Product List Parameters

Multipage: Can be set to '1' or '0' where 1 is on and 0 is off. This gives the page developer the ability to give the user the option to have a mutliple page product list. This is very handy on category pages. Home and Cart pages should almost always have multipage set to 0. Search results pages should ALWAYS have this set to 0, turning it on will cause the page not to function properly.

Default: This line contains a variety of 'default' behaviors.
Listing Style: Set to THUMB in the example above, can be any of these: PLAIN, THUMB, BIGTHUMB, THUMBMSRP, DETAIL, DETAIL2, MULTIADD, PLAINMULTI, SMALLMULTI, BIG, BIG2, PLAINMSRP. This determines what the default listing style is. Can be changed by user in the website buidler unless FORCEFORMAT=1 is set (see custom product lists for more information).
# of Columns: The first integer in the default determines the default number of columns that will be set to the product list. You can also force this number by using 'FORCECOLUMNS=1' (see custom product lists for more information).
Row Background Color: There are several options for setting the row background color.

Sort Order: Set to NONE in the example, this determines the default sorting order for the product list. If set to none, then the products will display in the order set in the product chooser in the website builder. Other options are ALPHABETICAL, PRICE, and RANDOM.
