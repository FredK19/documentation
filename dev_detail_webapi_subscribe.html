Merchant WebAPI subscribe.cgi

Purpose

This API provides a quick and easy way to add customers to the mailing list. It is designed to be transparently called by another CGI or Flash element because it returns XML rather than HTML. For an HTML version simply use the http://merchant.zoovy.com/subscribe.cgis form.

Parameters

merchant - the zoovy username
fullname - the fullname of the person subscribing e.g. "John Doe"
email - the email address of the person subscribing.

Responses

All responses are sent back in XML as follows:
<?xml version="1.0"?> <subscribe> .. contents .. </subscribe>
The contents will either be an <error> tag containing an error message, or a <success> tag containing a successful response.
