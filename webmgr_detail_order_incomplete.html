Order / Incomplete Items

Creating new Incomplete Items

By creating incomplete items you can let people check out and pay for items which were not listed through Zoovy. For example if a losing bidder contacts you after an auction ends and says they would like to be able to purchase items at the winning price and you agree, then you could simply create an incomplete item and the item would be processed just like any other item purchased through an auction.
'Creating Incomplete Sales'
  • From zoovy.com go to Orders
  • Select Incomplete
  • Use the Create New link at the bottom of the page.
  • Please note, this is not to create an order, but a link so your customer can checkout.
  • Fill in the information and the customer email address, select save then exit.
  • To determine the checkout link for the customer, open the incomplete item by the claim number link.
  • When you create an incomplete item, your customer is sent checkout and payment instructions for that item via email.

    • Finding Auction numbers

    Auction Numbers can be found by going to Incomplete items. The fastest way to find an incomplete item (closed auction) is to go to Orders | Search and plug in the auction id into the Incomplete item search.

    Non Paying Bidders

    Each marketplace has different policies when it comes to Non-paying bidders, you should refer to the channel details to learn more about how each marketplace handles non-paying bidders. The incomplete items screen lists which items have not been paid for. If you are using the eBay Unpaid Item dispute any incomplete items you have open disputes for will remain in the incomplete items folder of Order Manager (in either Web Manager or Zoovy Order Manager.)

    Sending Reminders to NPBs

    If you would like to remind your non paying bidders about coming to checkout, you can follow these steps:
    'Reminders'
  • Select the Orders tab
  • Select incomplete orders
  • Choose the checkbox for the incomplete item you want to send reminders for
  • Open the incomplete item and click send reminder
  • Edit the message for sending
  • Hit Save
