eBay Profile Tips

Overview

This webdoc lists several tips on setting up ebay profiles to help you list your products to eBay. It includes common use tips, along with a couple "gotchas". Bear in mind that eBay changes their requirements and configuration options quite often, and what works today, may not next week.
Since recent USPS rate changes these tips may be out of date. Please notify support if you should find one that no longer works. Also for further updates of known eBay issues please see Doc 50887

Tips for configuring your eBay profile

There are two links for every Profile: "Config" and "Shipping." There is also an HTML Wizard that must be selected. Be certain to setup both parts of your Profile.

Zoovy Rates vs. eBay Rates

Your eBay Profile uses shipping rates independent of rates that you have configured under Setup > Shipping. Your shipping rules, handling, insurance, etc., configured under Setup > Shipping have nothing to do with the shipping that you configure for eBay. EBay will obtain shipping rates based on the methods you have set in your profile.
If you have enabled zoovy:html you will not be able to modify the product description, when launching your items to eBay. You will need to use the product powertool to remove this attribute from the product and channel records. If you aren't familiar with using the product powertool contact support and have them remove this attribute for you and schedule some time with an implementation engineer for instructions on how to successfully use the product powertool.

EBay Gallery

This setting is often missed by merchants, then when the behavior performs as configured, but not as recalled, leaves the merchant believing that something is "broken". If you want gallery images for any type of listing, select that configuration setting. Likewise, if you don't want any listing to use gallery images, select that. If you set this to "Auction Only" and then do not see a galley image when you list a Fixed Price listing, you'll now know why.

Flat Rate Shipping

You must designate a method along with the flat rate for your items when using Flat Rate Shipping.
  • If you would like the flat rate to be based on a product by product basis, you must first set the price per product. You will then set the shipping cost / additional item cost to -1
  • If you would like to set a global flat rate for all products, simply input the desired price for shipping cost / additional item cost.

    • Free Shipping

    A convenient checkbox has been provided to enable Free Shipping. You must still select a method that corresponds to the Free Shipping offer.
    WARNING! These information is only a suggested method that has worked in the past. The information is VERY subject to change by Ebay as they frequently change their API in this area without notice. Try it, test it, and monitor it. If it DOESN"T work...adjust and test some more. Implementation or Tech Support may have current tips or tricks on getting free shipping to work on ebay, but don't take this functionality as a given.

    Error Codes

    At the top of the Shipping Profile page, you may see some error codes. Errors that begin with "Z-" are Zoovy errors. Usually, these can be corrected within your Zoovy account. Errors that start with "EBAY-" come from eBay. Often these are requirements that eBay asks for given the shipping methods that you have selected.
  • " Inconsistent tag values provided. To specify non-zero value for the tag you must set "true" for the tag."
    This error appears if you have "None" checked under shipping destinations. Uncheck "None."

    • Return Policy

    Currently Zoovy doesn't set the ebay listing "Return Policy". We do however allow you to set a return policy inside of your HTML Wizard that will get displayed in your ebay listing layout. To set this you can go to: Setup > Account Settings:Company Profiles, and click on the eBay Template name of the profile that you wish to adjust your policies.
