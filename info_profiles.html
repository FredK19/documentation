Profiles

Concept Overview

Profiles provide a mechanism for tracking separate company info. Configuration profiles store information such as contact information, about us, returns policy, email messages, and much more. Each profile may have a unique wrapper, pricing schedule, and root category, company logo, etc. You may have an unlimited number of profiles, and you may associate an unlimited number of domains to each profile. Products also reference profiles to load their policy information for transmission to marketplaces.

Company Profiles

Profiles can be used to create separate policies for different domains. This means you have the ability to have a different look and feel for each domain. Profiles can also be used so that you can launch to eBay with different policies.
How to Create a New Profile

  • Click on the company profiles link
  • Push the 'Add new Profile' button
  • Give the Profile a short Alphanumeric name/code
  • You can copy an existing profile by selecting it in the 'copy from' dropdown
  • Push the 'create profile' button

    Profile Pages

    Each company profile may have it's own Homepage, About Us, Contact Us, Privacy Policy, Return Policy, Shopping Cart Page and more.
    You can manage the page layout, images, description, meta tags, and product listings by clicking the edit link of the page you would like to edit.
    There is a known issue with specialty store homepages sharing the product with the homepage of your default profile. The work around for this is to create a list for the homepage of the specialty store and select that as the source. Doc 50325|filename explains what lists are and how to use them.

    Deleting Profile

    Profiles are like the cornerstone of a building, taking them away is rarely necessary, and typically dangerous." -- Brian Horakh, Zoovy System Architect
  • Removing profiles from the system is very dangerous. If the profile is still being referenced by a product or website the removing it can cause a enormously wide range of nasty business impacting issues ranging from minor problems such as missing contact information, to more severe blank emails and blank/corrupt listings, a large briar patch of issues.
  • It is your responsibility to make sure a profile is NOT in use prior to removing it. At a minimum this means making sure that no products are using the profile, and no domains use the profile, and no listings are live, or scheduled for any marketplace which has been created using this profile.
  • If you cannot say with absolute certainty that the profile is not in use please do NOT proceed!
  • Diagnosis of technical issues stemming from deleted profiles can be very difficult and resource intensive. If the removal of a profile is determined to be the cause you will incur additional fees and face possible loss of removal profile priviledges for repeat offenders.
  • It's also worth mentioning that most profiles consume less than 20k of disk space on your account (so 50 profiles = 1mb of space) so removing them to save space is really silly. If you want to save disk space, try removing images instead!
    Steps to Deleting a Profile
  • 1. Go to Setup
  • 2. Click on Company Profiles
  • 3. Click on "Remove Profile" button
  • 4. Go to "Profile to Remove" and select the profile you would like to remove
  • 5. Check the box for the disclaimer
  • 6. Click the "Remove Profile" button

    • Changing the HTML Wizard for your Profile

    To change the display layout for your profile, you need to edit/select the HTML Wizard that you desire to use in your listings.
    Steps to Edit/Select a HTML Wizard
  • 1. Go to Setup
  • 2. Click on Company Profiles
  • 3. Under "Configuration Profiles" click on the eBay Template name for the profile you would like to edit/select
  • 4. When the configuration page opened, either make desired changes to the current profile and "Save" (Preview or Exit), or....
  • 5. Click the "Change" box and select a different HTML Wizard to use for that profile.
