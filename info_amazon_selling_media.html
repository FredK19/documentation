[SECTION]Order Import]
Zoovy can import an Amazon Order File

[[HINT]Amazon Help]
Help Documentation:

https://sellercentral.amazon.com/gp/help/help.html?ie=UTF8&itemID=28501&language=en_US

[[/HINT]]

[[HINT]Sample File]
http://sellercentral.amazon.com/help/merchant_documents/reports/flat_file_order_report.xls
[[/HINT]]

To Import an Amazon Order File you need to replace the Amazon Header with a Zoovy Import Header. The following header is believed to work:

The file needs to have the header:
#GROUP_BY=%EMAIL
