Purpose: to allow download's of the stores SKU list.
Parameters: on GET pass USERNAME, PASSWORD, METHOD=|LIST|DOWNLOAD

METHOD=DOWNLIST

Also pass TIMESTAMP= (if not passed, then TIMESTAMP is assumed to be zero). Returns a list of all products which have changed since TIMESTAMP. Response Format:
