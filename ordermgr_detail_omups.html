Shipping with UPS World Ship

IMPORTANT NOTE:

Worldship is not a supported method of processing your orders. Please use the built in UPS Online Tools functionality within the Order Manager desktop client to process shipments.
While the synchronization process is a snap, there is some setup that requires the preparation of your shipping software to accept the exported files. Please consult your WorldShip Software Documentation for specific instructions on setting up your import/export settings.

About this Feature

The Zoovy Order Manager makes it easy to import tracking information and export customer shipping data to the UPS "WorldShip" software.
Exporting Invoices From Order Manager
  • 1. Run Zoovy order manager
  • 2. Select the invoices you would like to export
  • 3. Right Click one of the selected orders
  • 4. In the menu, select "Export to >> Shipping Export File" to save
  • 5. In UPS WorldShip, go to UPS online connect
  • 6. Select batch import
  • 7. Locate the file that you created and load
    • 'Import Tracking Numbers'
  • 1. Export your End of Day report from WorldShip
  • 2. With your file created, you will need to edit to use Zoovy headers as seen in this doc: Doc 50956
  • 3. Once done, open Order Manager
  • 4. Right Click within the order list screen (you don't need to select a specific order)
  • 5. In the menu, select "Import From >> Shipping import file"
    • Check your import file
    Verify that your import headers conform to Zoovy import standards. Check for spaces, strange characters, and verify casing matches what is displayed in the webdoc. Note that as soon as this file is imported, all orders listed will have tracking appended. Depending on your configuration, this may move your orders automatically to completed, and trigger your STATCOMP email. It's suggested that you create a template with the proper headers, and copy/paste your End of day information to avoid issue.

    Troubleshooting Tracking Import

    If you have trouble with importing tracking numbers, check to see whether the file is being recreated, or appending data to the file. Appending means that new items are added to the old with each import. This should not be happening. To update, follow these instructions.
    'Change Appending to Overwrite'
  • Run UPS Worldship
  • Go to UPS Online Connect in the menu system, then click on create/edit map
  • Choose the export map name and click the edit button
  • Change the radio button from appending the file to overwriting the file. and hit the okay.

    • Multiple Packages

    UPS (multiple boxes) option is used for split shipments. For example, if you have a customer that ordered three items that need to be shipped in two boxes, you would use this option. Keep in mind that when you use this the shipment will split according to SKU or product ID.

    Key Import with UPS

    The instructions for Key Import are as follows:
    'Key Import'
  • 1. Run UPS Worldship
  • 2. Go UPS Online Connect (in menu)
  • 3. Then go to Key Import
  • 4. Click on more then choose the named of import that was created for Zoovy.
  • 5. It will now prompt for Enter Import Key
  • 6. You can either scan the invoices or type the invoice number.
  • 7. Process shipment as normal

    • Comma delimited imports convert the dashes (-) to underscores (_) for the invoice number. If you type in an invoice number, you will need to enter underscore as the invoice number. For example 2003-05-2569 needs to be input as 2003_05_2598.
