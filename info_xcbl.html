XCBL Order Format

Concept Overview

xCBL is a common platform neutral format that zoovy can export order data into. xCBL is used by a variety of EDI gateway systems such as eBridgeSoft.com

API Dispatch

] When xCBL orders are dispatched to an HTTP or HTTPS (recommended) URL then the payload is transferred in the "Contents" field. Effectively the same as if the following code was posting it:
<form action="" method="POST"> <textarea name="Contents">xcbl payload goes here</textarea> </form>
