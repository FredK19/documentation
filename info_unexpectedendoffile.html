Unexpected end of file has occurred

Symptom:

Upon trying to Sync or close Order Manager the user gets an Exception Error.
The details of the error will be something similar to this, however details may vary:
************** Exception Text **************
System.Xml.XmlException: The 'Package' start tag on line 101 does not match the
end tag of 'ToCompany'. Line 102, position 3.

Solution

This is normally the result of a corrupted Dazzle xml file. The solution is to find the file that is corrupt and remove it.
Here are the steps to do this.
Finding Corrupt XML File
  • Close Order Manager
  • Open "My Computer" and navigate to C:\program files\Zoovy\Desktop\tracking
  • You will see files in here that have an extension of .XML. It is one of these files that is corrupt.
  • Create a sub folder called temporary and a subfolder called corrupted.
  • Move all of the .XML files from the tracking folder into the temporary subfolder.
  • Now move one of the .XML files back into the tracking folder.
  • Attempt to open Order Manager.
  • If Order Manager works fine, close it. Repeat steps 6 and 7 until Order Manager works.
    • Once you know which file is causing the error, copy that into the Corrupted subfolder. You can then copy the rest of your .XML files back in to the tracking folder one at a time. Test each one to make sure they are all ok.
    You will need to reprocess the orders that are associated with the corrupted .XML file.
