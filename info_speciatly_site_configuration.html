Specialty Site Configuration

Concept Overview

Specialty Sites are a way for a merchant to have multiple online web stores that use the same checkout, shipping, payment methods, order processing, etc., as the primary store. The configuration is fairly simple, but there are a few steps that you will have to take before you can start populating your site.

Setup Categories

You will want a different set of navigation categories for your new site. This is done by creating a hidden category to represent your new site. Then create subcategories beneath that one to be the navigation that is displayed.
Category creation steps
  • Click Setup
  • Click Categories & Lists
  • Click Add New (beneath Homepage)
  • Click Create a name for the category, and use an exclamation point as the first character. That will ensure that the category is hidden and does not show up on your primary store.
  • Click Save
  • Expand the new category and click Add New to begin creating your subcategories.

    • Company Profile

    You will need to create a company profile that is specific to this site.
    Creating Profile
  • Click Setup.
  • Click Domain Hosting.
  • Click the sub-tab Profiles and Partitions.
  • Create a name for the new profile in the field marked 'Code:.'
  • Click Create Profile.

    • Domain Name

    The next thing you need to do is register a domain name. This will be the domain that you will use as the URL for your specialty site.
    Registration steps
  • Click Setup
  • Click Domain Hosting
  • Click Add Domains
  • Click Register
  • Enter the domain name that you want to register. (make sure the name is available before registering.)
  • Click Register
    • Once the domain is registered you will need to configure it to point to your new company profile.
    Domain / Profile configuration
  • Click Setup.
  • Click Domain Hosting.
  • On the sub-tab Domain Config, expand the domain name section you're modifying and click on Edit Configuration.
  • Under Hosting Type select "Specialty Site Hosting."
  • Under Configuration Profile select the new company profile that you created earlier.
  • Click Save Configuration.

    • Configuring Profile

    Now it is time to finish the configuration.
    Setting up the new store profile
  • Click Setup
  • Click Site Builder. You will now see a configuration box for your new profile.
  • Expand the configuration box
  • Click Edit for Company Information
  • In the Root Category drop down box select the new hidden category that you created earlier
    • The rest of the this profile can be configured the same way as the original.
