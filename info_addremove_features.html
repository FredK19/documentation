Add/Remove Features

Concept Overview

If you need to add or remove a bundle, flag or feature to your Zoovy account, you will need to go to Setup Add/Remove Features. Before doing this, please schedule an implementation appointment for assistance. Implementation appointments can be scheduled by logging in to your account and going to Support > Request Implementation Assistance.
You will not be able to remove the feature that are the basis of your account. For some Zoovy users, this will be the Auction Bundle. For other Zoovy users, this will be the Website Bundle.
Be careful! Do not remove a feature unless you are absolutely certain that you do not need it.
How to add or remove a feature
  • Go to Setup > Add/Remove Features.
  • Find the feature that you are interested in modifying.
  • Click on either "Add" or "Remove."
