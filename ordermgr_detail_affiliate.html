Affiliate Report

Summary

The affiliate report will show gross sales for a range, or an entire status based on the !META property of an order.

The !META can be set by passing an additional parameter ?META=CODE to any page on your zoovy site. For example:
http://yourname.zoovy.com?META=CODE http://www.yourdomain.com?META=CODE
In this case CODE would appear as a column in the report. You can make up new codes using characters A-Z and 0-9.
