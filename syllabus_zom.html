Syllabus: Order Manager

ZOM Training

This training will go over the initial basic steps for using Zoovy Order Manager.

PREREQUISITES

Before your training session, please download and install the latest version of Zoovy Order Manager. It is found in your Zoovy account under the Downloads link.
You will also need to download and install a current version of the .NET platform. You can find a link to Microsoft's .NET download page on the same page that you download Order Manager.

ZOM Training Agenda

You cannot log into Zoovy Order Manager until you have created a User Login. You can find your User logins under Setup > Manage User Login.

  • Logging into ZOM.
  • The Sync button.
  • Changing order status in ZOM.
  • Printing and modifying invoices.
  • Automatic email messages.
    Once you sync an order with Zoovy Order Manager, you want to use Order Manager to modify it from then on. Modifying orders online and in Order Manager may result in lost or corrupt data.
