XML Order Format

This is the format of an order when it is sent via an XML dispatch to a remote API. The order listed below is for a Paypal Order, however other fields such as masked credit card will be present if the order is paid by credit card. For specific detail on the formatting of the order, payment status codes, and more please contact zoovy developer support.
