Supply Chain STOCK

STOCK Overview

The supply chain integration currently allows you to easily add inventory and/or subscribe to direct product feeds from suppliers. More information about the entire Supply Chain Featureset can be found: Doc 50694
Step by Step process for Stock Order Dispatches
  • Step 1 - product's inventory decremented
    - customer buys product from your website (or Amazon, ebay, etc) OR
    - csv inventory import OR
    - manually thru the Product Editor
    If you have Stock Supplier Items that have been created with zero (0) inventory and a minimum stock qty greater than zero, these items will NOT be automatically added to your Stock Order. (ie inventory must change to alert the system that these items should be ordered.) In this case, go to Zoovy: Utilities: Supply Chain Management: STOCK Supplier: Inventory Update Configuration and press UPDATE.
  • Step 2 - inventory decrement is finalized
    - script finalizes this inventory decrement
    - script analyzes in-stock qty and min stock qty
    - (this process can be long, 6hrs+)
  • Step 3 - inventory finalization script adds Supplier Order Item (as appropriate)
    - inventory finalization finds in-stock qty lower than min stock qty
    - adds supplier order item, status of item is NEW
  • Step 4 - script adds order item to Supplier Order
    - if an existing order is OPEN, item gets added to order
    - if not, a new order is OPENed
  • Step 5 - limits are analyzed
    - days of the week, cost minimum and max days for open order are all analyzed to determine if order is ready to be dispatched
    - Order is dispatched/emailed to supplier when limits are met
