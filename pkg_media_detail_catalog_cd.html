CD Catalog

Overview

The book catalog contains information indexed by UPC. When uploading to the CD catalog you must use the manufacturer assigned UPC.

Fields

Field Name Required Description
UPC Y This field must contain a valid UPC or UPC/A barcode value.
Attribute: zoovy:prod_upc
Product Format Y The list of available formats is maintained by Zoovy. Currently the following CD Formats are supported: CD, CD Single, CD Compliation (contains more than one artist)
Attribute: zoovy:prod_format
Product Language Y The list of available CD languages is maintained by Zoovy.
Attribute: zoovy:prod_language
CD Title Y The album title as it appears on the compact disc.
Attribute: zoovy:prod_name
Description Y Text about the album.
Attribute: zoovy:prod_desc
CD Cover Y An image of the front of the CD - minimum: 75x50 maximum: 640x400
Attribute: zoovy:prod_image1
Artist Y The band or individual which recorded the album. If the CD is a compilation please put "Various"
Attribute: zoovy:prod_artist
Label/Studio N The studio which produced the album (e.g. Geffen, Columbia)
Attribute: zoovy:prod_label
Release Date N Format: MM/DD/YYYY if (month and/or day are not known put 00/00/YYYY)
Attribute: zoovy:prod_release
Tracks N The number of tracks the CD contains (note you can leave this blank if you also supply tracks, we'll figure it out)
Attribute: zoovy:prod_tracks
Advisory N If the CD contains explicit lyrics as defined by the RIAA. (possible values 0/1)
Attribute: zoovy:prod_advisory
Genre (Unstructured) N The unstructured genre contains a category for the cd. For example "Pop", "Rock", "Jazz", or "Classical" NOTE: At this time Zoovy is in the process of developing a structured Genre field based on feedback from sellers.
Attribute: zoovy:prod_ugenre
List of Tracks N This field should contain each track, in order, one per line.
Attribute: zoovy:prod_tracklist
Reviews N A list of reviews (ideally formatted with HTML) for the particular product.
Attribute: zoovy:prod_detail
