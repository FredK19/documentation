Order Manager Installation Guide - version 6.x

Overview - Order Manager Installation Guide

This webdoc provides a walk through on installing Order Manager version 6.012 and above on your local system.

Before You Begin

There are several factors to consider and prepare for before you begin installation.

Supported Operating Systems:

Windows 2000 Service Pack 3; Windows Server 2003; Windows XP Service Pack 2, in addition to all 64 bit and IA64 bit Windows versions.

Before Installing:

All applications require the Microsoft .NET 3.5 framework to be installed for proper operation. Please visit: Microsoft .Net Framework Download Site
If you are not sure which version of the .NET framework is right for your computer use the x86 Redistribute Package version. However if you have a 64 bit machine, then you can download the x64 for more faster performance - consult your IT manager for details.
Windows Update
Zoovy Desktop software will only operate on computers which are fully patched via Windows Update. Please visit: http://windowsupdate.microsoft.com

Installing Order Manager

Download a copy of Zoovy Order Manager from your Web Account Manager at http://www.zoovy.com/biz/downloads/. Save the program to your computer. You need to stay up to date with the latest version, so remember to upgrade.

First Time Installations

For a first time installation, just download the files through your web manager from the download link. Double click on the icon from your desktop and follow the installation instructions.

Upgrades with Windows 9X, Me or 2000

If you have Windows 9X, Me or 2000, just download the latest release from your web manager and install the upgrade over the previous version. Make sure you have the latest release to take advantage of added features.

Upgrades with Windows XP

If you having difficulty with Windows XP and you are installing an upgrade, please follow the steps below. The steps for proper upgrade installation on Windows XP:
  • 1. Uninstall the older version of Order Manager
  • 2. Rename the Order Manager folder located on your hard drive in this directory c:\Program Files\Zoovy\Ordermgr to C:\program files\zoovy\ordermgr.old
  • 3. Reboot the computer
  • 4. Install the new version of Order Manager
